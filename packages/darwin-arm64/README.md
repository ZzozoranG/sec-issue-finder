# @zzozorang/sec-issue-finder-darwin-arm64

Prebuilt `sec-issue-finder` binary package for macOS arm64.

This package is intended to be installed as an optional dependency of `@zzozorang/sec-issue-finder`. It is not meant to be used directly.

Expected binary path:

```text
bin/sec-issue-finder
```

The binary is not committed in source control. Release builds will place it here before packaging.
