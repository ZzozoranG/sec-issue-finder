# Code of Conduct

[한국어](CODE_OF_CONDUCT.ko.md)

## Our Pledge

We are committed to a respectful, harassment-free project environment for contributors and users.

## Expected Behavior

- Be respectful and constructive.
- Focus criticism on ideas and code, not people.
- Assume good intent while being clear about technical concerns.
- Respect maintainer decisions about scope and release readiness.

## Unacceptable Behavior

- Harassment, threats, or abusive comments.
- Personal attacks.
- Publishing private information without permission.
- Repeatedly derailing technical discussions.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or restrict participation when behavior is inconsistent with this code of conduct.

Report conduct concerns privately to the repository maintainers.
